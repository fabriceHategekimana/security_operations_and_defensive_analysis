# Volatility

## Télécharchez et installer Volatility 2: 

### prerequisite

- git (optionel)
- python2

### À executer:

```bash
1. pip install --upgrade setuptools
2. sudo apt-get install python2-dev
3. pip2 install pycrypto && pip install distorm3
```
